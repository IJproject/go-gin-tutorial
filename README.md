### バックエンド

- Gin(Golang) 

### フロントエンド

- Nuxt(TypeScript)
- Vuetify

### インフラ

- Docker
- Docker Compose

#### やることリスト

- モデルを導入（gorm？）

- Docker環境では、フロントエンドとバックエンドのどちらでもホットリロードができるように構築しており、快適に開発が進められるようにしています。
